HTML5 e CSS

HTML5

  HTML5 é a quinta versão da linguagem HTML, a linguagem mais recente de marcação de hipertexto usada para criar páginas, sites, aplicativos da web.
  Esta nova versão traz consigo importantes mudanças quanto ao papel do HTML no mundo da Web, através de novas funcionalidades como semântica e acessibilidade.

    •      		✔️ - Indica o menu de navegação com os links para as páginas

    Foi utilizado dentro do 'body' depois do 'nav' para que a informação seja o principal e único.

    CSS

      CSS , é uma linguagem de estilo, adicionando estilos (cores, fontes, espaçamento, etc.) a uma página web em HTML e outras linguagens de marcação.
      Pode ser aplicado diretamente nas tags HTML ou ficar contido dentro das tags 'style', ou adicionando um link para um arquivo CSS que contém os estilos. Assim,
    quando quiser alterar a aparência, a ilustração dos documentos vinculados a este arquivo CSS, basta só modifica-lo.

    Tipos de seletores CSS   ✔️/🧐   Descrição
  • universal(*)
    •      		✔️ - Aplica o estilo a todos os elementos da página.

    Foi utilizado em todas as página, para mudar o tipo de letra.
  • tipo
    •      		✔️ - Usa o nome do marcador a estilizar.

    Foi utilizado para estilizar os marcadores do tipo h1,h2...

  • classe
    •      		✔️ - Associa um nome a um estilo.

    Foi utilizado em há várias partes que serão estilizados num só tipo.Neste caso dentro do body.

  • id
    •      		✔️ - Permite identificar univocamente um elemento do HTML (muito útil no JavaScript).
    Neste o seletor, o identificador (case-sensitive) é antecedido de #.

    Foi utilizado nas palavras HTML5(h4) e CSS(h4) para facilmente identificar um único conteúdo a estilizar (mudar de cor),

  • atributo
    •      		✔️ - Aplica em atributos de elementos, pode ser apenas referido o atributo (target), pode-se também fazer depender do valor do atributo

    Foi utilizado na parte onde está o 'nav' depois de cada um dos links pertencentes ao sua prórpia página. Ao clicar na página pretendida os outros ficam de outra cor.

  • pseudo-classe
    •      		✔️ - É associado ao estado de um elemento. É uma palavra-chave adicionada a seletores que especifica um estado especial do elemento selecionado
    (por exemplo: link – ao elemento antes de ser clicado; visited – ao elemento depois de clicado; hover – quando o rato passa por cima do elemento...etc).

    Foi utilizado para configurar cores nos links quando ,antes ou depois de clica-lo.

  • pseudo-elemento
    •      		✔️ - É uma palavra-chave adicionada a um seletor que permite estilizar uma parte específica do elemento selecionado.

    Foi utilizado para aumentar a primeira letra de cada parágrafo.

    Combinadores de seletores ✔️/🧐   Descrição
  • Seletores descendentes
    •      		✔️ 'div p' – todos os p que estão “dentro” de um div

    Foi utilizado muitas vezes quando tenho um h1, h2.. etc para eles terem uma melhor organização.
  • Seletores filhos
    •      		✔️ 'div > p'– todos os p que estão “logo dentro” de div

    Foi utilizado mais para o final desta página.
  • Seletores adjacentes
    •      		✔️ 'div ~ p'– todos os p que estão colocados depois de um div

    Foi utilizado na parte do quizzz em que entre uma pergunta e a outra tendo um 'p'

  • Selet. imediatamente adjacentes
    		• 🧐 'div + p'– todos os p que estão colocados “imediatamente” depois de um div

    Por enquanto ainda não foi utilizado.
  • Agrupar
    •      		🧐 'div,p{color:red;}'- os div e p ficacarão com fonte vermelha

    Por enquanto ainda não foi utilizado.
